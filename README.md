# CocoDoogy_Scripts
Unity3D를 활용한 모바일 퍼즐게임 6인 팀프로젝트입니다.

본 저장소는 저작권이 포함된 외부 에셋을 제외하고, 핵심 게임 로직을 담당하는 스크립트 위주로 구성된 Public Clone입니다.

## 플레이 과정 및 기능
### Auth
1. Firebase를 이용한 구글로그인 또는 게스트 로그인
2. 유저데이터가 없으면 닉네임 설정 팝업

### Lobby
1. 친구 창을 통해 친구 추가 및 티켓 주고받기로 게임 입장 횟수 추가 가능
2. 상점 창을 통해 아이템, 티켓 및 교환을 위한 유료 재화 구매 가능
3. 설정 창을 통해 마스터, 배경음, 효과음 세 가지의 볼륨 조절
4. 프로필 창을 통해 내 닉네임 및 스테이지 클리어 진도 확인
5. 배경 스와이프를 통해 어떤 테마의 스테이지들을 플레이할지 선택
6. Start 버튼으로 선택된 테마의 스테이지 선택창으로 입장

### Stage Select
1. 좌우 스크롤을 통해 원하는 스테이지 선택
2. 첫 번째는 기본적으로 열려있고, 클리어 여부에 따라 하나씩 개방
3. 열려있는 스테이지 선택 시 해당 스테이지에서 나오는 기믹들에 대한 간단 정보 팝업 제공
4. 선택된 스테이지 입장 전 소모 행동력/소모 시간에 따른 랭킹과 현재 보유 중인 아이템 보유 갯수도 확인 가능
5. Start 버튼을 통해 선택된 스테이지 내부로 입장

### Ingame
1. 여러 효과의 날씨 속에서 타일을 클릭해 밟아가며 이동
2. 타일 또한 다양한 종류가 존재하며, 주어진 행동력 소진 시 이동 불가
3. '돌아가기' 클릭 시 스테이지마다 주어진 횟수를 소모하여 행동력과 위치를 리셋함 (실행했던 기믹은 유지됨)
4. 목적지에 도착 시 클리어이며 '돌아가기'와 '행동력' 모두 소진 시 실패
